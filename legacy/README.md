# Archived campaigns

This directory preserves superseded scientific work so that rejected ideas
remain auditable without appearing in the active workflow.

- `counterterms/`: microscopic, projected, and dressed winding-counterterm
  scans.  They left residual winding transfer and are not the production
  method.
- `masked_bw/`: parity-masked Brillouin-Wigner experiments and the previous
  "any coupling" narrative.  The construction lacked the required canonical
  order/grid convergence campaign.
- `development_notes/`: pedagogical figures, exploratory observables, and
  earlier perturbative checks not imported by active code.
- `preliminary_submission/`: the earlier FCC-32/Ce2Hf2O7 submission package.
  Its FCC-32 spectra were order-three ice-manifold results, not a
  full-Hamiltonian material fit.

Archived scripts may depend on their original paths and are retained as
records rather than supported entry points.
