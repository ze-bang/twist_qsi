# Twist-averaged ED for quantum spin ice (16-site demo)

Demonstration of twist-averaged exact diagonalisation as a topologically
clean replacement for "counter-term" corrections to spurious finite-size
ring exchanges in QSI ED on small pyrochlore tori.

## Layout

```
twist_qsi_demo/
├── paper/
│   ├── main.tex              <- standalone LaTeX writeup (theory + results)
│   ├── main.pdf
│   └── figs/                 <- all figures, generated by analyse scripts
├── scripts/
│   ├── twist_helper.py            <- writes Trans.dat / InterAll.dat with twist
│   │                                  on cubic-conventional clusters
│   ├── fcc_cluster_helper.py      <- same writer for FCC-primitive clusters
│   │                                  (the (2,2,2) 32-site recommended geometry)
│   ├── run_demo.py                <- 8-corner sweep (FTLM + Lanczos)
│   ├── twist_sweep_1d.py          <- continuous phi_x scan (Lanczos)
│   ├── analyze_demo.py            <- post-processing + figure generation
│   ├── projector_kernels.py       <- partial-vs-full projector kernel demo
│   ├── cluster_geometry_audit.py  <- enumerates contractible vs wrapping
│   │                                  hexagons / 4-cycles per cluster shape
│   │                                  (cubic AND fcc bases)
│   └── plot_cluster_geometry.py   <- builds fig_cluster_geometry.pdf
├── output/
│   ├── demo/                 <- 8-corner HDF5 + summary.json
│   └── sweep1d/              <- 13-point phi_x HDF5
└── README.md
```

## Reproducer

From the repo root, after building the C++ ED binary:

```bash
cd twist_qsi_demo
python3 scripts/run_demo.py --out output/demo \
    --ftlm-samples 12 --ftlm-krylov 150 --n-eigs 64
python3 scripts/twist_sweep_1d.py
python3 scripts/analyze_demo.py
cd paper && pdflatex main.tex && pdflatex main.tex
```

Wall time on a 16-thread workstation: ~6 min for the 8-corner sweep, ~22 s
for the 1D scan, instantaneous post-processing.

## Physical setup

- 1x1x1 pyrochlore cluster, 16 sites, 48 NN bonds, 8 tetrahedra
- XXZ couplings: `Jxx = Jyy = 0.2`, `Jzz = 1.0` => `Jpm = -0.1`, `Jpmpm = 0`
- Twist grid: 8 corners `(0, pi)^3` for the main demo; finer 13-point
  `phi_x in [0, 2*pi)` for the photon dispersion picture
- Hilbert space: full `2^16 = 65536` (no Sz projection, FTLM and Lanczos
  on the unprojected Hamiltonian)

## What the figures show

- `fig_specific_heat.pdf` -- bare vs twist-averaged C(T) (the central
  result). Bare ED has a sharp spurious low-T peak at T ~ 0.04 from the
  8-fold near-degenerate finite-size ground multiplet; twist averaging
  spreads and lowers it; the high-T ice-rule peak is unchanged.
- `fig_specific_heat_diff.pdf` -- C_bare - C_twist plus the corner
  spread, showing where the finite-size correction lives.
- `fig_entropy.pdf` -- entropy per site, with the Pauling ice plateau
  and ln(2) limits.
- `fig_spectrum_vs_twist.pdf` -- lowest 48 eigenvalues at each twist
  corner, exposing the 8-fold spurious multiplet and its twist
  dispersion.
- `fig_lowband_vs_phix.pdf` and `fig_photon_dispersion.pdf` -- the
  spurious photon dispersion made visible via continuous twist.
- `fig_cluster_geometry.pdf` -- (a) per-site contractible-hexagon
  density `nu_c` (= 1 on every faithful cluster: the hexagonal
  coupling `g_hex = 12|Jpm|^3/Jzz^2` is preserved at the bulk strength
  after twist averaging), (b) smallest non-zero cluster wavevector
  `k_min` with and without twist refinement, and (c) the corresponding
  lowest accessible photon energy compared to the bulk Schottky scale
  `T_peak = 0.42 g_hex`. Includes both cubic-conventional clusters
  (1,1,1)..(2,2,2) and FCC-primitive clusters (2,2,2)..(4,2,2). The
  FCC (2,2,2) 32-site cluster preserves the full pyrochlore O_h
  symmetry and is the recommended production geometry. See
  `paper/main.tex` Sec. 6 for the full discussion.

  Note: FCC primitive shapes with `min(L1, L2, L3) < 2` -- e.g.
  (1,1,1)_fcc (4 sites), (2,1,1)_fcc (8 sites), and (2,2,1)_fcc
  (16 sites) -- are *geometrically degenerate*: the cluster
  fundamental domain is shorter than 2 NN spacings in some
  direction, so the NN graph is not faithful (some sites have <6 NN
  partners and several site pairs have multiple minimum-image NN
  bonds). `fcc_cluster_helper.py` refuses to build them by default;
  use `(1,1,1)_cubic` for a 16-site ED experiment. See
  `paper/main.tex` Sec. 6.2 for the derivation.
